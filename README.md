# 📚 AI Course Generator

An AI-powered platform that automatically generates personalized courses, learning roadmaps, chapter-wise content, and educational resources from a simple topic prompt.

The goal of this project is to make learning faster, smarter, and more accessible by using Artificial Intelligence to create structured educational content instantly.

---

## 🚀 Features

* 🤖 AI-Generated Course Creation
* 📖 Automatic Chapter Generation
* 🧠 Personalized Learning Roadmaps
* 🎯 Topic-Based Course Structure
* 📚 Dynamic Educational Content
* 🎥 Resource & Video Recommendations
* ⚡ Instant Course Generation
* 📱 Responsive Modern UI
* 🔍 Smart Learning Path Suggestions
* 🌐 Real-Time AI Assistance

---

## 💡 How It Works

Simply enter a topic such as:

> "Machine Learning"

or

> "Web Development"

The AI automatically generates:

* Course Overview
* Learning Objectives
* Chapter Structure
* Detailed Topics
* Learning Resources
* Study Roadmap
* Recommended Progression Path

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/Pranshu51/Ai-Course-Generator.git

cd Ai-Course-Generator
```

---

### 2. Install Dependencies

```bash
npm install
```

---

### 3. Configure Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_key

CLERK_SECRET_KEY=your_key

DATABASE_URL=your_database_url
```

---

### 4. Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
